  • htmlEscape(allowUnsafeSymbols:decimal:encodeEverything:useNamedReferences:)

    Return string as HTML escaped by replacing non-ASCII and unsafe characters with their numeric character escapes, or if such exists, their HTML named character reference equivalents. For example, this function turns "<script>alert("abc")</script>" into "&lt;script&gt;alert(&quot;abc&quot;)&lt;/script&gt;" View/set options globally via String.HTMLEscapeOptions. - parameter allowUnsafeSymbols: Specifies if all ASCII characters should be skipped when escaping text. Optional - parameter decimal: Specifies if decimal escapes should be used instead of hexadecimal escapes. Optional - paramter encodeEverything: Specifies if all characters should be escaped, even if some are safe characters. Optional - parameter useNamedReferences: Specifies if named character references should be used whenever possible. Optional
